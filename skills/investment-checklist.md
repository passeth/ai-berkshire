# 버핏 가치 투자 매수 전 Checklist

$ARGUMENTS에 대해 버핏 가치 투자 매수 전 Checklist 분석을 실행합니다.

**입력 형식 지원**: 단일 또는 복수 회사, 쉼표/공백으로 구분. 예: `텐센트, 마오타이, 엔비디아` 또는 `NVDA AAPL MSFT`

## 실행 흐름

### 1단계: 입력 파싱, 분석 대상 회사 식별

$ARGUMENTS에서 모든 회사명/코드를 파싱합니다. 각 회사에 대해 결정:
- 회사 전체 이름, 종목 코드, 상장 거래소
- 상장되지 않은 회사인 경우 "비상장"으로 표시하고 간단한 설명(간접 투자 경로 존재 여부) 제공, 전체 Checklist는 생략

### 1단계 반: AI 연구 편향 경고

각 회사에 대해 "정보 풍부도" 빠른 등급(A/B/C)을 매기고 보고서에 표시:

| 등급 | 판단 기준 | Checklist에 미치는 영향 |
|------|-----------|-------------------------|
| A급 | 상장 다년, 데이터 충분 | 정상 실행하되 "컨센서스 함정" 주의 — 모든 지표가 명확해 보인다고 진짜 확실한 것은 아님 |
| B급 | 데이터 제한적, 추정 필요 | 각 추정 지표에 신뢰도 표기, "좋은 사업" 판단 시 데이터 신뢰성을 가중 고려 |
| C급 | 정보 극도로 희박 | 6단계 표를 억지로 채우지 말고, "데이터 부족으로 판단 불가"를 솔직히 표기, 검증 가능한 핵심 질문에 집중 |

**핵심 원칙**: Checklist의 목표는 **나쁜 선택을 배제**하는 것입니다. C급 회사에 대해 "데이터 부족"은 "불통과"도 "통과"도 아닙니다 — "회색지대, 1차 정보 보완 필요"라고 솔직히 표시해야 하며, AI가 표를 채울 수 없다고 해서 거부로 판단해서는 안 됩니다.

단용평이 말했습니다: "모르겠다"에는 두 가지가 있다 — 사업이 너무 복잡해서 진짜 모르는 경우, 아직 시간을 들여 보지 않은 경우. AI 연구의 한계는 "자료가 적다"와 "모르겠다"를 쉽게 혼동하는 것입니다.

### 2단계: 병렬 데이터 수집

**각 회사마다** 독립적인 백그라운드 Agent를 Task 도구로 시작해 데이터 수집을 수행합니다(모든 회사를 동시에 병렬 시작). 각 Agent는 다음을 수집합니다:

1. **수익성**: ROE (5-10년 추이), 매출총이익률, 순이익률, 잉여현금흐름
2. **밸류에이션 데이터**: 현재 주가, 시가총액, PE(TTM), 선행 PE, PB, 배당수익률
3. **성장 추이**: 최근 3년 매출/이익 증가율
4. **재무 건전성**: 부채 수준, 자본지출 수요, 현금 보유, 순현금/순부채
5. **경쟁 구도**: 시장점유율, 주요 경쟁사, 점유율 변화 추이
6. **해자 증거**: 브랜드/전환비용/네트워크 효과/규모 효과/기술 장벽의 구체적 증거
7. **경영진 기록**: CEO 이력, 주요 의사결정, 지분, 자본 배분 기록
8. **최신 동향**: 최근 6개월 주요 이벤트(실적, M&A, 규제, 경영진 변동 등)

### 3단계: 회사별 6단계 Checklist 실행

상장된 각 회사에 대해 순차적으로 6단계를 통과시킵니다:

---

#### 1단계: 이 사업을 이해할 수 있는가 (능력권)

반드시 답해야 합니다:
- [ ] 한 문장으로 이 회사가 어떻게 돈을 버는지 설명할 수 있는가?
- [ ] 10년 후에도 대체로 같은 사업을 할 가능성이 높은가?
- [ ] 성패를 좌우하는 핵심 변수는 무엇인가?
- [ ] 이 업계에 대한 인식은 깊이 있는 연구에서 왔는가, 아니면 풍문에서 왔는가?

**평가 기준** (★1-5):
- ★★★★★: 비즈니스 모델이 극도로 단순하고 명확하며, 10년 확실성이 높음 (예: 마오타이 — 술 빚어 파는 것)
- ★★★★☆: 모델은 명확하나 기술적 장벽이 있어 일정 수준 전문 지식 필요
- ★★★☆☆: 모델은 이해 가능하나 10년 확실성이 높지 않고, 업계 변화가 빠름
- ★★☆☆☆: 사업 라인이 복잡하거나 업계가 급변 중이어서 미래 예측 어려움
- ★☆☆☆☆: 완전히 능력권 밖

**강제 거부**: 돈 버는 방식조차 설명할 수 없으면 "능력권 밖, 분석하지 않음"으로 즉시 표시.

---

#### 2단계: 좋은 사업인가 (경제적 특성)

데이터로 말하며, **핵심 지표는 반드시 도구로 정확 계산**해야 합니다:

```bash
python3 ~/ai-berkshire/tools/financial_rigor.py verify-valuation \
  --price {주가} --eps {EPS} --bvps {주당순자산} --fcf-per-share {주당FCF} --dividend {주당배당}
```

| 지표 | 해당 회사 수치 | 참고 기준 | 판단 |
|------|----------------|-----------|------|
| ROE (5년 평균) | | >15% 우수, >20% 탁월 | |
| 매출총이익률 | | >40%는 가격 결정력 시사 | |
| 잉여현금흐름 | | 지속 플러스, ≈순이익 | |
| 자본지출 강도 | | 경자산 우수, 중자산 대비 | |
| 부채 수준 | | 유이자부채/순이익 <3년 | |

**평가 기준** (★1-5):
- ★★★★★: ROE>25%, 고마진, 강한 FCF, 경자산, 저부채 (모두 충족)
- ★★★★☆: 4개 항목 충족
- ★★★☆☆: 3개 항목 충족
- ★★☆☆☆: 2개 항목 또는 추이 악화
- ★☆☆☆☆: 대부분 미달 또는 FCF 지속 마이너스

---

#### 3단계: 해자가 충분히 깊은가 (경쟁우위)

항목별 점검:

| 해자 유형 | 보유 여부 | 구체적 증거 | 넓어지는가 좁아지는가? |
|-----------|-----------|-------------|------------------------|
| 브랜드/가격 결정력 | | | |
| 전환 비용 | | | |
| 네트워크 효과 | | | |
| 비용/규모 우위 | | | |
| 기술/특허 장벽 | | | |

추가 검증: 경쟁사에게 100억을 준다면 이 사업을 복제할 수 있는가?

**평가 기준** (★1-5):
- ★★★★★: 다중 해자 중첩 + 더 넓어지는 중
- ★★★★☆: 최소 1개의 강력한 해자 + 안정적
- ★★★☆☆: 해자는 있으나 충분히 깊지 않거나 추이 불명확
- ★★☆☆☆: 해자가 침식되고 있음
- ★☆☆☆☆: 뚜렷한 해자 없음

---

#### 4단계: 경영진을 신뢰할 수 있는가 (사람 요인)

| 점검 항목 | 평가 |
|-----------|------|
| 정직성 (약속 vs 이행) | |
| 자본 배분 능력 (자사주 매입/배당/M&A 기록) | |
| 주주 이익 지향 (지분 보유, 보수) | |
| 소유자 마인드 (창업자 vs 전문 경영인) | |
| 지배구조 (관련자 거래, 영업권, 감사) | |
| CEO가 떠나도 정상 운영 가능한가? | |

**평가 기준** (★1-5):
- ★★★★★: 창업자가 키를 쥐고 있으며, 자본 배분 탁월, 이해관계 완전 일치
- ★★★★☆: 경영진 우수하나 작은 하자 있음
- ★★★☆☆: 경영진은 적격하나 지배구조 리스크 존재
- ★★☆☆☆: 정직성 또는 지배구조 문제 있음
- ★☆☆☆☆: 심각한 정직성 문제 (강제 거부)

---

#### 5단계: 가격이 충분히 싼가 (안전마진)

| 지표 | 수치 | 역사적 분위 | 판단 |
|------|------|-------------|------|
| PE (TTM) | | | |
| 선행 PE | | | |
| PB | | | |
| 배당수익률 | | | |
| FCF Yield | | | |

추가 검증 (**반드시 도구로 정확 계산, 암산 금지**):
```bash
python3 ~/ai-berkshire/tools/financial_rigor.py three-scenario \
  --price {주가} --eps {EPS} --shares {발행주식수 억} \
  --growth {낙관} {중립} {비관} --pe {낙관PE} {중립PE} {비관PE} --currency {통화}
```
- 3시나리오별 밸류에이션 구간 (도구 출력 결과 사용)
- 만약 판단이 틀렸을 때 현재 가격에서 매수하면 최대 얼마까지 손해 볼 수 있는가?
- 주가가 반토막 나도 추가 매수할 용기가 있는가?

**평가 기준** (★1-5):
- ★★★★★: 내재 가치 대비 5할 이하, 극단적 안전마진
- ★★★★☆: 7할 수준, 양호한 안전마진
- ★★★☆☆: 합리적 밸류에이션, 안전마진 보통
- ★★☆☆☆: 비싼 편, 안전마진 부족
- ★☆☆☆☆: 심각하게 고평가

---

#### 6단계: 비중과 의사결정 규율 (감정 통제 방지)

다음 감정 신호를 점검:
- FOMO 때문에 사고 싶은가?
- 남이 추천해서 사고 싶은가?
- 5년간 거래 정지되어도 견딜 수 있는가?
- 매수 논리를 200자 이내로 명확히 쓸 수 있는가?

### 4단계: 거울 테스트

각 회사에 대해 거울 테스트 문장을 작성:

> "나는 ___원에 ___회사를 매수한다. 이유는 다음과 같다:
> 1. 이 사업의 본질은 ___이며, 나는 이를 이해한다.
> 2. 해자는 ___이며, 더 넓어지고 있다 / 좁아지고 있다.
> 3. 경영진은 ___하며, 신뢰할 수 있다 / 없다.
> 4. 현재 가격은 내재 가치의 ___할 수준으로, 충분한 안전마진이 있다 / 없다.
> 5. 설령 내가 틀렸더라도 하락 리스크는 통제 가능 / 불가능하다. 이유는 ___ 때문이다."

**5문장으로 설명이 안 되면 사지 않는다.** "통과" 또는 "미통과"를 명확히 표시.

---

### 5단계: 신속 거부 목록

각 회사에 대해 항목별 점검. 하나라도 해당되면 즉시 "거부" 표시:

- [ ] 이 회사가 어떻게 돈을 버는지 설명할 수 없음
- [ ] 3년 연속 잉여현금흐름 마이너스이며 개선 조짐이 없음
- [ ] 경영진에 정직성 오점 있음
- [ ] 경쟁우위가 되돌릴 수 없이 침식되고 있음
- [ ] "다음에 더 비싸게 사줄 사람"에 의존해 돈을 벌려 함 (바보 게임)
- [ ] 이 투자가 0이 되어도 감당할 수 없음
- [ ] 매수 이유가 주로 "남들도 사고 있다" 또는 "최근 많이 올랐다"임
- [ ] 매수 이유를 200자 이내로 명확히 쓸 수 없음

---

### 6단계: 총괄 비교표 출력 (다중 회사 분석 시 필수)

여러 회사를 분석할 때는 반드시 총괄 비교표를 생성:

| 회사 | Checklist 통과? | 능력권 | 좋은 사업 | 해자 | 경영진 | 안전마진 | 핵심 결론 |
|------|-----------------|--------|-----------|------|--------|----------|-----------|
|      |                 | ★☆☆☆☆  | ★☆☆☆☆     | ★☆☆☆☆ | ★☆☆☆☆  | ★☆☆☆☆    |           |

---

### 7단계: 최종 결론 및 파일 작성

각 회사에 대해 명확한 결론 제시 (회피하지 않음):
- ✅ **Checklist 통과** (X/6단계) — 심층 연구 단계 진입 가능
- ❌ **Checklist 미통과** — 어떤 레드라인이 트리거되었는지 설명
- ❓ **회색지대** — 핵심 쟁점이 무엇인지 설명, 투자자가 스스로 판단해야 할 부분 명시
- N/A — 비상장 / 매수 불가

전체 보고서를 `~/버핏Checklist-[회사명 또는 "다중회사비교"].md`에 작성.

## 출력 형식 요구사항

1. 각 회사는 독립 챕터로 구성: 6단계 평가표 + 핵심 데이터표 + 주요 리스크 (3-5개) + 거울 테스트 + 명확한 결론
2. 다중 회사 분석 시 마지막에 총괄 비교표 첨부
3. 모든 평점은 ★ 기호 사용 (★1-5, 반별 없음)
