# 재무제표 정독 팀 + 공중계 게시: 4대 거장 병렬 해석 → 편집 → 독자 리뷰

$ARGUMENTS 에 대해 재무제표 정독 팀 분석을 실행하고, 필요 시 공중계 게시용 아티클로 가공합니다.

입력 예: `PDD 2025년 연례 보고서`, `텐센트 2025Q4`

## 팀 구성

- **Team Lead**: 총괄, 종합 판단, 최종 보고서/아티클 편집
- **4대 거장 에이전트**:
  - 단용평 관점: 비즈니스 모델과 수익 품질
  - 버핏 관점: 현금흐름, 이익의 지속성, 안전마진
  - 멍거 관점: 리스크, 역발상, 실패 시나리오
  - 리루 관점: 장기 트렌드 적합성, 문명적 맥락

## 실행 흐름

1. 4개 Agent 병렬로 동일 재무제표 세트(원본 연례/분기 보고서 + 실적 발표회 의사록)를 읽고 각 관점에서 분석.
2. 각 Agent는 숫자 + 경영진 발언을 직접 인용하며 "이 숫자가 의미하는 것"을 자신의 관점으로 해석.
3. Team Lead가 4개 보고서를 종합하여 하나의 일관된 "재무제표 정독 보고서" 작성.
4. (선택) 공중계 게시 버전으로 편집: 
   - 독자 친화적 서두 추가
   - 어려운 회계 용어 설명
   - "한 줄로 요약" + "핵심 숫자 5개" 박스
   - "투자자 관점 시사점" 섹션
5. 독자 리뷰 모드: "이 아티클을 읽은 후 가장 큰 질문 3가지"를 가상으로 생성하여 보완.

## 출력

- `reports/[회사]-[기간]-earnings-team.md` (전체 분석)
- 공중계 버전 별도 파일 (선택)
- "게시 전 최종 체크리스트" (데이터 정확성, 가독성, 결론 명확성)

모든 숫자는 원본 1차 자료에서 직접 인용하고, 2차 요약은 사용하지 않습니다.
